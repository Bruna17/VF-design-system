# Logo

## Horizontal

```html
<div class="vf-logo-container">
    <div class="vf-logo">
        <img src="assets/images/visitfinland.png" alt="logo">
        <div class="vf-logo-text">Visit Finland</div>
    </div>
</div>
```

```html
<div class="vf-logo-container">
    <div class="vf-logo">
        <img src="assets/images/visitfinland-black.svg" alt="logo">
        <div class="vf-logo-text vf-logo-text-black">Visit Finland</div>
    </div>
</div>
```

Remove the black background before use.

```html
<div class="vf-logo-container">
    <div class="vf-logo" style="background-color: #000;">
        <img src="assets/images/visitfinland-white.svg" alt="logo">
        <div class="vf-logo-text vf-logo-text-white">Visit Finland</div>
    </div>
</div>
```
